# Match Care ABA — GitHub Pages Deployment Guide

## What's in this folder

```
matchcareaba-github/
├── index.html        ← Your website
├── family-hero.jpeg  ← Family photo
└── DEPLOY.md         ← This file
```

---

## Step 1 — Create a Free GitHub Account

1. Go to https://github.com
2. Click **"Sign up"** — use any email
3. Choose a username (e.g. `matchcareaba`)
4. Verify your email

---

## Step 2 — Create a New Repository

1. Once logged in, click the **"+"** in the top right → **"New repository"**
2. Name it exactly: `matchcareaba.github.io`
   *(This is important — the name must end in .github.io)*
3. Set it to **Public**
4. Click **"Create repository"**

---

## Step 3 — Upload Your Files

1. On the repository page, click **"uploading an existing file"**
2. Drag both files into the upload area:
   - `index.html`
   - `family-hero.jpeg`
3. Scroll down and click **"Commit changes"**

---

## Step 4 — Enable GitHub Pages

1. Go to your repository **Settings** (top tab)
2. Click **"Pages"** in the left sidebar
3. Under **"Source"**, select **"Deploy from a branch"**
4. Branch: **main** / Folder: **/ (root)**
5. Click **Save**

✅ Your site will be live at:
**https://matchcareaba.github.io**

Takes about 2-3 minutes to go live after saving.

---

## Updating the Site in the Future

When you get a new zip from Claude:

1. Unzip it
2. Go to your GitHub repository
3. Click on `index.html` → click the **pencil/edit icon** → then click the **upload icon**
4. Upload the new file → **Commit changes**

Or drag and drop new files directly into the repository.
